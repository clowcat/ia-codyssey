# 퀴즈 게임 (Quiz Game)

## 1. 프로젝트 개요
이 프로젝트는 Python 기초 문법과 객체 지향 프로그래밍(OOP)을 활용하여 제작한 콘솔 기반의 퀴즈 게임 프로그램입니다. 사용자가 퀴즈를 풀고 점수를 기록하며, 새로운 문제를 직접 추가하여 나만의 퀴즈 데이터를 구축할 수 있는 기능을 제공합니다.

## 2. 퀴즈 주제 선정 이유
- **주제**: 과일
- **이유**: 남녀노소 누구나 쉽게 이해할 수 있는 친숙한 주제이며, 사물에 대한 명확한 특징(색상, 모양, 계절 등)이 있어 4지선다형 문제를 구성하기에 적합하여 선정하였습니다.

## 3. 실행 방법
프로그램을 실행하기 위해서는 **Python 3.10 이상** 버전이 필요합니다.
1. 터미널(또는 CMD)을 열고 프로젝트 폴더로 이동합니다.
2. 아래 명령어를 입력하여 프로그램을 실행합니다.
   ```bash
    python main.py
   ```

## 4. 기능 목록
- **퀴즈 풀기**: 저장된 퀴즈를 순차적으로 출제하며, 정답 여부를 즉시 확인하고 최종 결과를 표시합니다.
- **퀴즈 추가**: 사용자가 직접 문제, 4개의 선택지, 정답 번호를 입력하여 새로운 퀴즈를 등록합니다.
- **퀴즈 목록**: 현재 시스템에 등록된 전체 퀴즈의 질문 목록을 확인합니다.
- **점수 확인**: 역대 최고 점수(가장 많이 맞힌 정답 수)를 확인합니다.
- **종료 및 저장**: 프로그램을 종료할 때 추가된 퀴즈와 최고 점수를 `state.json` 파일에 안전하게 저장합니다.
- **예외 처리**: 숫자 변환 실패, 범위 밖 숫자, 빈 입력 등에 대한 예외 상황을 처리하여 비정상 종료를 방지합니다.

## 5. 파일 구조
- `main.py`: 프로그램의 메인 로직, `Quiz` 클래스 및 `QuizGame` 클래스가 포함된 소스 코드 파일
- `state.json`: 퀴즈 데이터와 최고 점수가 저장되는 JSON 형식의 데이터 파일
- `README.md`: 프로젝트 설명 및 사용 가이드 문서

## 6. 데이터 파일 설명 (state.json)
프로젝트 루트의 `state.json` 파일은 UTF-8 인코딩으로 데이터를 관리하며, 아래와 같은 스키마 구조를 가집니다.

| 키(Key) | 타입(Type) | 설명 |
| :--- | :--- | :--- |
| **`quizzes`** | Array (List) | 등록된 퀴즈 객체들의 목록. 각 객체는 `question`, `choices`(배열), `answer`(정답 번호)를 포함함 |
| **`best_score`** | Integer | 사용자가 기록한 역대 최고 정답 개수 |
